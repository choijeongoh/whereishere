# whereishere
whereishere은 1개월 동안 준비한 팀프로젝트입니다.
6개월동안 배운 것을 총 활용해서 작업한 프로젝트이며, 3명의 팀원과 함께 만들었습니다.

# 프로젝트 소개
 ﻿가입한 업체들의 정보를 확인하고 이벤트 및 회원들의 평가를 공유할 수 있는 웹 사이트이다.
